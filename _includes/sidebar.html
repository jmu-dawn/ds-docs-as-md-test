{%- comment -%} Builds the docs sidebar by grouping every page under docs/ by its SOURCE subfolder. Group order follows `nav_group_order` in _config.yml; any group not listed there is appended afterwards in natural order. Each subfolder's heading label comes from a page's `group:` front matter, else the folder name. Order within a group is controlled with `nav_order:` (lower = higher). {%- endcomment -%} {%- assign doc_pages = site.pages | where_exp: "p", "p.path contains 'docs/'" -%} {%- assign groups = doc_pages | group_by_exp: "p", "p.path | remove: p.name" -%} {%- comment -%} Precompute each group's label into an array parallel to `groups`. {%- endcomment -%} {%- assign labels_str = "" -%} {%- for group in groups -%} {%- assign first = group.items | first -%} {%- assign segs = group.name | split: "/" -%} {%- if first.group -%} {%- assign label = first.group -%} {%- elsif segs.size <= 1 -%} {%- assign label = "Overview" -%} {%- else -%} {%- assign label = segs | last | replace: "-", " " | replace: "_", " " | capitalize -%} {%- endif -%} {%- assign labels_str = labels_str | append: label | append: "§" -%} {%- endfor -%} {%- assign labels = labels_str | split: "§" -%} {%- assign order = site.nav_group_order -%}
{%- comment -%} 1) Groups named in nav_group_order, in that order. {%- endcomment -%} {%- for order_label in order -%} {%- for label in labels -%} {%- if label == order_label -%} {%- assign g = groups[forloop.index0] -%} {%- include sidebar_group.html group=g label=label -%} {%- endif -%} {%- endfor -%} {%- endfor -%} {%- comment -%} 2) Any remaining groups not listed in nav_group_order. {%- endcomment -%} {%- for label in labels -%} {%- unless order contains label -%} {%- assign g = groups[forloop.index0] -%} {%- include sidebar_group.html group=g label=label -%} {%- endunless -%} {%- endfor -%}
